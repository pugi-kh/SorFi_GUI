# SorFi

파일 정리 GUI 프로그램.

![SorFi Screenshot](resource/screenshot/스크린샷%202025-10-03%20032248.png)

## 설치

Python 미설치 시 `01_python_check` 실행 → MS Store에서 설치

## 실행

```bash
python src/SorFi_main.py
```

또는 `02_RUN` 실행

## 주요 기능

- 폴더 경로 지정
- 폴더명 생성 (직접 입력 / 자리수 지정)
- 문자열 위치 확인 옵션
- 설정 저장/불러오기 (JSON)
- 되돌리기

## CLI 버전

터미널에서 사용하는 간단한 버전: [MerFi](https://github.com/pugi-kh/MerFi)
